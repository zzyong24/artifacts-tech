# 《让创意更有黏性》— 提纲稿

### Slide 1：开场钩子

→ 那种感觉：明明懂，但说不清楚 {step}
→ 对方懵 → 你说"算了，他不懂" {step}
→ 错。不是他不懂，是你不会说 {step}
→ 懂得越多，越说不清楚 {step}
→ 科学实验证明的 {step}

---

### Slide 2：知识的诅咒

→ 斯坦福实验：一组人敲节奏，另一组猜歌名 {step}
→ 敲的人预测猜对 50%，实际：2.5% {step}
→ 敲的人脑里有旋律，对方只听到"哒哒哒" {step}
→ 这叫「知识的诅咒」——知道了，就永远无法想象不知道 {step}
→ 沟通的头号杀手 {step}

---

### Slide 3：原则一 — 简单

→ 怎么破？第一个原则：简单 {step}
→ 不是说得浅，是把核心观点打磨成一句谚语 {step}
→ 西南航空案例：加鸡肉沙拉？→ "我们是全球票价最低的航空公司" {step}
→ 就这一句，所有争论结束 {step}
→ 你现在有没有那一句话？ {step}

---

### Slide 4：原则二 — 意外

→ 第二个原则：意外 {step}
→ 大脑只对变化敏感，对一成不变自动屏蔽 {step}
→ 广告案例：温馨家庭出游 → 侧面撞车 → 黑屏 → "没料到吧？系好安全带" {step}
→ 永远刻进脑子了 {step}
→ 你上次的 PPT，有没有让人"没想到"的时刻？{step}
→ 全是"众所周知"，就是在帮观众催眠 {step}

---

### Slide 5：原则三 — 具体

→ 第三个原则：具体。知识诅咒最直接的解药 {step}
→ "高性能" vs "V8 发动机"——哪个懂？ {step}
→ "世界顶级客户服务" vs "取车前帮你把车预热好" {step}
→ 感官能感知到的，就是具体的 {step}
→ 说不清楚时：不要解释，去举例；不要讲道理，讲故事 {step}
→ 伊索寓言流传两千年，靠的是狐狸和葡萄，不是论文 {step}

---

### Slide 6：原则四＋五 — 可信 + 情感

→ 可信 + 情感，放一起说 {step}
→ 数字会杀死同理心——反直觉 {step}
→ 实验：数据 vs 罗基娅（7岁小女孩的故事）→ 故事捐款翻倍 {step}
→ 同时展示数据+故事 → 比只看故事还少 {step}
→ 数据启动分析大脑，把情感关掉了 {step}

---

### Slide 7：原则六 — 故事

→ 最后，也是最强的武器：故事 {step}
→ 听故事时大脑不是被动接收，是在跑模拟器 {step}
→ 好故事 = 行动前的脑内演练 {step}
→ 三种结构：挑战情节（激励）/ 联系情节（合作）/ 创造情节（创新） {step}

---

### Slide 8：金句收尾

→ 六个原则：SUCCESs {step}
→ 底层逻辑只有一句：你以为说清楚了，对方只听到了"哒哒哒" {step}
→ 克服知识诅咒，不是让你变笨，是记住：你面前的人是一张白纸，不是你 {step}
