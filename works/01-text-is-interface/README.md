# 01 · 文本是知识的接口

> 教学短视频 · 口播稿 + 交互动画

## 内容简介

讲清楚一件事：在 LLM 时代，文本不是落后的内容形式，而是知识的通用接口。
面向普通用户，类比驱动，短句口播风格。

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 交互式教学动画，9 张幻灯片，GSAP 驱动 |
| `script.md` | 口播稿，含每页时长标注、封面标题、话题标签 |
| `gsap.min.js` | GSAP 动画库本地文件（避免跨域问题）|

## 使用方式

```bash
# 在项目根目录启动本地服务（见根目录 README）
python3 -m http.server 8080

# 浏览器打开
open http://localhost:8080/works/01-text-is-interface/index.html
```

## 操作说明

| 操作 | 动作 |
|------|------|
| 点击 / 空格 / → | 下一步动画 / 下一张 |
| ← | 上一张 |
| 全屏录制 | F11 或 Cmd+Ctrl+F，配合 OBS / QuickTime |
