01
成本鸿沟
视频转文本：免费，成本近零
多模态 AI 直接处理：成本翻几十倍，还有时长 / 次数限制

能长期坚持的 = 最稳定、成本最低的
02
隐性知识的抓手
没有文本框架，视频里的微表情、言外之意 = 玄学

文本是知识导航图：先搭好显性框架，再带着导航去看视频，才能精准捕捉隐性信息

文本：网状、可编辑、可随时重组

阶段 1
收录：把信息转化为可处理的文本
🔗
ThirdSpace
  • 一键收录文章 URL
  • AI 自动生成知识卡片
  • 摘要 + 关键要点
  • 三个深度思考问题（连接 / 挑战 / 行动）
🎬
清记 NoteClear
  • 支持 B站 / YouTube / 抖音 / 小红书 / 本地文件
  • 有字幕视频：几十秒出结构化 Markdown 笔记
  • 拿到文字后 → 走同样的知识卡片路径
  • 视频和文章，最终统一为同一种格式
三个深度思考问题（非「你怎么看」式废话问题）
连接层
这个知识，和你自己的哪段经历对应？
挑战层
这里面有没有你不同意的地方？
行动层
如果只能改变一件事，你会改变什么？
阶段 2
反思：让知识真正进入你的认知
Mirror
把你分散的思考回答，提炼成一个核心认知主题
Deepen
再给三个更深的问题——行为层 · 情绪层 · 创造层
Bridge
洞察 → 具体行动建议 → 自动写入待办清单 + 附原始链接
核心设计：没有被执行验证的行动，不算真正的闭环
阶段 3 + 4
输出 + 立体化：同一套认知，两种感官传递
✍️ 阶段 3：输出文章
  • 知识库已积累大量文本素材
  • AI 扫描主题，提炼重组草稿
  • 你的工作：审阅 / 修改 / 补充真实经历
  • 文章骨架从积累里长出来，不是强行拼凑
🎬 阶段 4：立体化为视频
  • 同一篇文章 → 两个互补维度
  • 口播稿：听觉 × 情感节奏（为「说」优化）
  • 教学动画：视觉 × 结构关系（以原文为内容源）
  • 提词器与动画通过 BroadcastChannel 自动联动
这不是一篇文章变成两个内容，而是同一套认知，用两种感官同时传递
