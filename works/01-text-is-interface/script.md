# 文本是知识的接口 — 提词器稿

> 作品编号：02-sop-to-skills
> 主题：LLM时代，文本是知识的通用接口

---

### Slide 1：你是不是也有过这种感受

→ 刷了一下午知识视频，转头就忘
→ 收藏夹存满干货，再也没打开
→ 追 AI 学习工具，越学越焦虑

---

### Slide 2：我们搞错了什么？

→ 以为是不够自律、方法不对
→ 更根本的问题：搞错了知识的**载体**？ {step}

---

### Slide 3：文本不是落后的内容形式

→ 文本 = 知识的**通用接口** {step}
→ 接口 = 连接、互通、可二次加工、持续产生价值
→ 视频/音频 = 传播形式；文本 = 交互接口 {step}

---

### Slide 4：第一层 — 你和知识的对话接口

→ 刷视频 = 被动的线性接收 {step}
→ 跟着博主节奏，无法拆解、无法关联
→ 当时懂了，转头就忘。从来没真正对话过 {step}
→ 文本天生为主动对话而生：可暂停/标注/拆解/重组
→ 视频让你**看见**知识，文本让你**走进**知识 {step}

---

### Slide 5：第二层 — AI 的原生接口

→ 大模型的「母语」= **文本** {step}
→ 总结/推理/对比/体系搭建，全基于文本训练
→ 喂视频给 AI：先转文本 → 信息损耗 + 成本翻几十倍 {step}
→ 喂干净文本：直接对接 AI 原生能力 {step}

---

### Slide 6：第三层 — 现在和未来的迭代接口

→ 收藏的视频 = **一次性内容**：下架、大海捞针 {step}
→ 文本 = **能穿越时间的活接口** {step}
→ 一秒定位 / 补充新认知 / 修正旧观点 / AI 跨期对比
→ 打通现在的你和未来的你 {step}

---

### Slide 7：为什么是普通人的最优解

→ **多模态 = 富人方案，文本 = 普惠选择** {step}
→ 1小时视频：转文本免费 vs 多模态成本翻几十倍
→ 能长期坚持 = 成本最低、最稳定 {step}
→ 没有文本框架，视频隐性知识 = 玄学
→ 文本是**知识导航图**，先框架再看视频 {step}

---

### Slide 8：一套完整的工作流

→ 我自己实践：文本贯穿全程 {step}
→ 收录 → 反思 → 输出 → 立体化 {step}

---

### Slide 9：阶段1 — 收录

→ 文章：ThirdSpace 一键收录 → **知识卡片** {step}
→ 三个深度思考问题：连接层 / 挑战层 / 行动层
→ 从「收藏了」→「被问到了」= 主动思考 {step}
→ 视频：清记转文字（B站/YouTube/抖音）→ 同路径
→ 最终：统一文本节点，可检索/标注/追问 {step}

---

### Slide 10：阶段2 — 反思

→ 收录是起点，大多数人把它当终点
→ 真正消化 = 回答三个问题之后 {step}
→ 触发深层反思：Mirror / Deepen / Bridge
→ Bridge 自动写入待办，附原始链接 {step}
→ 核心设计：**没有执行验证的行动，不算闭环** {step}

---

### Slide 11：阶段3+4 — 输出 + 立体化

→ 知识库积累够了 → AI 从素材提炼重组草稿
→ 我只需：审阅 / 修改 / 补充真实经历 {step}
→ 文章写完 → **立体化为视频**
→ 口播稿：听觉 + 情感节奏；教学动画：视觉 + 结构关系 {step}
→ **同一套认知，用两种感官同时传递** {step}

---

### Slide 12：文本这道门

→ 整条链路：每个环节信息都以文本流动 {step}
→ 维特根斯坦：「我的语言的界限，就是我的世界的界限」
→ 文本不是多写字，是**让所有知识经过文本这道门** {step}
→ 经过文本的知识：AI能处理 / 你能内化 / 未来有价值
→ 现在就做：把触动你的点，用自己的话写下来 {step}
